Core Competencies

Foundation for Sites

I first learned of Foundation for Sites when I began working on networksolutions.com and have been using it ever since. If you're not familiar, think of Foundation as a family of responsive front-end frameworks that make it easy to design beautiful responsive websites, apps and emails that look amazing on any device. Foundation is maintained by Zurb Foundation and is an open source project.

jQuery: Write More. Do Less.

jQuery is a JavaScript library designed to simplify HTML DOM tree traversal and manipulation, as well as event handling, CSS animation, and Ajax. It is free, open-source software using the permissive MIT License. Web analysis indicates that it is the most widely deployed JavaScript library by a large margin.

Adobe Target

I first began working with this tool back in 2010 when I began working on register.com, and have been using it pretty much ever since then. There are 6 websites that I've worked on which use this tool: register.com, networksolutions.com, web.com, logoyes.com, 1shoppingcart.com, and leads.com. I've worked on over 525 campaigns between between each of these brands, each with its own objectives and configured success metrics.

Java Server Pages (JSP)

I have over 8 years of experience with advanced applications using JSP. JavaServer Pages is a technology that helps software developers create applications that contains dynamic data. Front end development on a Java Application is about combining multiple technologies in order to achieve a single goal, Along with JSP, I have also learned Struts, Tiles, Jenkins, CVS, SVN, and Eclipse.

Node js

Node.js is an open-source, cross-platform JavaScript run-time environment that executes JavaScript code outside of a browser. This website, kevinnolan.me was built on Node.js. I learned how to use Node and it's server, Express to develop the website on my local machine prior to making it live.

The World Wide Web Consortium (W3C)

As a developer furiously obsessed with writing clean and syntactically correct code, W3C has long since been used as a key ingredient in my finished work. The W3C was founded in 1994 by Tim Berners-Lee, inventor of the World Wide Web. I find the W3C Validator useful when developing custom UI components. Finished projects can be validated by url or code snippets can be pasted, both methods are useful depending on the project.
